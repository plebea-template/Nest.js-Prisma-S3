# Description

Template for a Nest.js server with Prisma and S3

# Bootstrap

## Dependency Installation

```
npm i -g pnpm
pnpm install
```

## Build

```
pnpm run build
```

# Start

## Production mode

```
pnpm run start:prod
```

## Development mode

```
pnpm run start:dev
```
